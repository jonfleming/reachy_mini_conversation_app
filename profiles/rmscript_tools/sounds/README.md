# rmscript sounds

Drop audio files here to use them from an rmscript tool with `play <name>` or
`loop <name>`. The name in the script is the filename **without extension**;
`.wav`, `.mp3`, and `.ogg` are searched, in that order.

Example: a file `cheer.wav` in this folder is played by `play cheer`.

## Built-in robot sounds

These ship with the SDK and can be played by name without adding a file here
(this folder takes priority if a same-named file exists):

- `confused1`
- `count`
- `dance1`
- `go_sleep`
- `impatient1`
- `wake_up`
